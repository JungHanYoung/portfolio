귤메이트

귤메이트는 가족의 일정 및 할일들을 관리할 수 있는 앱입니다. Spring Boot를 이용해 API 서버, AWS로 DB, 서버 인프라를 구축하였고 Android와 iOS 둘다 구축할 수 있도록 Flutter로 앱을 구현하는 두가지 작업을 병행하였습니다.

기능 하나하나는 간단하게 구현을 하였지만 서비스에서 기본적으로 지켜야할 것이라 생각되는 인증, 데이터 CRUD, API 연동, 파일 업로드, 실시간 채팅 등을 중점으로 개발을 진행하였습니다. 더 나아가 AWS EC2, RDS, S3로 클라우드 서버 및 DB를 구축하였으며 API 서버는 도메인을 구입하고 HTTPS 설정까지 하며 시작부터 배포까지의 과정을 경험하였습니다.
